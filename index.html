Privacy Policy – Bnoud App

Last Updated: … / … / 2025

At Bnoud App, we value your privacy and are committed to protecting user data in compliance with Google Play policies. This Privacy Policy explains how information is collected, used, and protected, including the use of analytics tools.

By using Bnoud App, you agree to this Privacy Policy.

1. Information We Collect

a) Information Provided by the User

  • Name
  • Phone number or email address
  • User type (Finishing Engineer / Freelancer / Company)
  • Project data (items, costs, payments, notes)
  • Financial data manually entered by the user

b) Information Collected Automatically

  • Device type and operating system
  • Device identifiers or analytics identifiers
  • App usage data (sessions, screen views, interaction data)
  • IP address (used for security and analytics purposes)

2. Analytics and Tracking Tools

Bnoud App uses third-party analytics services including Firebase Analytics (Google), Google Analytics for Firebase, and Microsoft Clarity.

These services collect aggregated and anonymized usage data to analyze performance, understand user behavior, and improve features. These tools do not access project content or sensitive financial data.

3. How We Use Information

  • Provide and operate core services
  • Manage accounts and projects
  • Improve performance and usability
  • Monitor crashes and technical issues
  • Communicate updates and support
  • Prevent fraud and misuse

4. Data Storage and Security

  • Data is stored using reasonable technical safeguards
  • Measures are taken to prevent unauthorized access
  • Users are responsible for protecting login credentials

5. Data Sharing with Third Parties

We do not sell user data. Limited data may be shared only with analytics providers, legal authorities when required by law, or trusted technical service providers under strict confidentiality obligations.

6. Project Data and Content

All project-related data is fully owned by the user. Content is accessed only for technical support upon user request.

7. App Permissions

  • Internet access (data sync and analytics)
  • File or photo access (uploading project documents)

8. User Rights

  • Update personal information
  • Delete account and associated data
  • Request details about stored data

9. Delete Account and Data

Users have the right to request deletion of their account and all data associated with the Bnoud App at any time through one of the following methods:

  • Contacting our support team via email:
    bnoudtech@gmail.com
    Account deletion requests are processed within a maximum of 7 business days.
  • Or submitting an account deletion request through the following link:
    https://bnoud.com/ar/remove_my_account

Once an account deletion request is submitted, the account and all related data — including any linked sub-accounts — will be permanently deleted. This action is irreversible and cannot be undone.

10. Children’s Privacy

Bnoud App is not intended for children under 18 years of age.

11. Changes to This Policy

This Privacy Policy may be updated periodically. Users will be notified of significant changes via the app or this page.

12. Contact Us

Email: bnoudtech@gmail.com

باستخدامك لتطبيق بنود، فإنك توافق على الشروط الواردة في هذه السياسة.

1. المعلومات التي نقوم بجمعها

أولًا: المعلومات التي يقدّمها المستخدم

  • الاسم
  • رقم الهاتف أو البريد الإلكتروني
  • نوع المستخدم (مهندس تشطيبات / فريلانسر / شركة)
  • بيانات المشاريع (البنود، التكاليف، الدفعات، الملاحظات)
  • البيانات المالية التي يُدخلها المستخدم يدويًا

ثانيًا: المعلومات التي يتم جمعها تلقائيًا

  • نوع الجهاز ونظام التشغيل
  • معرّفات الجهاز أو التحليلات
  • بيانات استخدام التطبيق
  • عنوان IP لأغراض الأمان والتحليل

2. أدوات التحليلات وتتبع الاستخدام

يستخدم تطبيق بنود خدمات تحليلات تابعة لأطراف ثالثة تشمل Firebase Analytics وGoogle Analytics وMicrosoft Clarity، بهدف تحليل الأداء وتحسين تجربة المستخدم.

3. كيفية استخدام المعلومات

  • تشغيل التطبيق وتقديم خدماته
  • إدارة الحسابات والمشاريع
  • تحسين الأداء وتجربة المستخدم
  • تحليل الأعطال
  • التواصل والدعم الفني
  • منع الاحتيال

4. تخزين وحماية البيانات

  • استخدام وسائل حماية تقنية مناسبة
  • منع الوصول غير المصرح به
  • مسؤولية المستخدم عن بيانات الدخول

5. مشاركة البيانات

لا نقوم ببيع بيانات المستخدمين. قد تتم مشاركة بيانات محدودة لأغراض التحليل أو الامتثال القانوني فقط.

6. بيانات المشاريع والمحتوى

جميع بيانات المشاريع ملك للمستخدم، ولا يتم الاطلاع عليها إلا بطلب المستخدم لأغراض الدعم الفني.

7. صلاحيات التطبيق

  • الوصول إلى الإنترنت
  • الوصول إلى الملفات أو الصور

8. حقوق المستخدم
